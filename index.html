Choose KnockOut Number:

Result: 0

Chances: 5 of 5

Game instructions

  1. First step is to choose the target number.
  2. Throw the dice until you get the targeted number.
  3. Until the targeted number is achieved the total score of the throws will be calculated.
  4. Once targeted number is thrown the game is over.
  5. The game is ON until you hit the Reset button which will be reseting the game to zero.
